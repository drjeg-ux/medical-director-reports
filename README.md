# Medical Director Report APP

A web-based application for PostAcuteNC to track and generate Medical Directorship Billing Reports.

## Features

- Provider & Facility management with email addresses
- 7 task categories for medical director duties
- Manual report creation with task logging
- **Auto Report Generator** with:
  - Visit date selection
  - QAPI meeting tracking
  - Survey participation tracking
  - Even task distribution across visit dates
  - Variation check to avoid identical reports
- Draft vs Complete report workflow
- PDF download with full task descriptions
- Email report functionality
- Password-protected Settings
- Report History with view/edit/delete/email actions

## Usage

Open `index.html` in any modern web browser. All data is stored locally in your browser's localStorage.
